# Functional Requirements: Weather-Driven Task Suggestion

- **Goal**: A dashboard that fetches real-time weather and suggests a productivity task.
- **Requirement 1**: Display current temperature and weather condition.
- **Requirement 2**: If temperature > 20°C, suggest "Outdoor Coffee Break/Walk".
- **Requirement 3**: If temperature <= 20°C, suggest "Deep Work / Focus Session".
- **Requirement 4**: All logic must be covered by Unit Tests (Vitest).
- **Requirement 5**: The UI must be verified by E2E Tests (Playwright).
