# Agentic Workflow Capstone

## Project Overview

A Next.js dashboard built using agentic workflows and TDD.

- **Architecture:** Multi-repo setup using Git Submodules.
- **Feature:** Weather-aware task recommendations.

## Setup Instructions

1. Clone the parent repo: `git clone --recursive <link>`
2. Navigate to `dashboard-app` and run `npm install`.
3. Run tests: `npm test` and `npx playwright test`.
