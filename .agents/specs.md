# Agent Role & Rules

You are an autonomous Senior Software Engineer.
We are building a Capstone project using Next.js and MCP.

## Project Standards

- **Workflow:** Always follow TDD. Write the test in `tests/` before the implementation.
- **Context:** Every feature must have a functional spec in the `context/` folder.
- **Tech:** Next.js 16+, TypeScript, Tailwind CSS.
- **Testing:** Vitest for unit tests, Playwright for E2E.

## MCP Configuration

- Use the Next.js MCP server to navigate the codebase and optimize routes.
